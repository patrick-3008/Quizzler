### Quizzler App using UIKit

This is a simple single-page quiz app built with UIKit, designed to test your knowledge on a variety of topics. It provides a straightforward user interface and is a great starting point for anyone interested in iOS app development with UIKit.

## Features

1. Single-page interface for a seamless user experience.
2. Multiple-choice questions with correct answers.
3. Randomized questions to keep the quiz interesting.
4. Score tracking to monitor your progress.
5. Clear and concise user feedback for each question.

# Getting Started

Follow these steps to get the quiz app up and running on your iOS device or simulator.

Prerequisites
Xcode installed on your macOS.
Basic knowledge of Swift and UIKit.
Installation
Clone or download this repository to your local machine.
bash
Copy code
git clone [https://github.com/yourusername/single-page-quiz-app.git](https://github.com/patrick-3008/Quizzler.git)
Open the project in Xcode.
Build and run the app on your iOS device or simulator.
Usage

Upon launching the app, you will be presented with the quiz page.
Read each question carefully and select your answer by tapping the corresponding choice.
After answering all the questions, tap the "Submit" button to see your score.
You can retake the quiz by tapping the "Restart" button.
Customization

You can customize this quiz app to suit your needs by modifying the following:

Question Data: Update the questions and answers in the QuizBrain.swift file.
User Interface: Modify the UIKit components and layout in the Main.storyboard file.
Scoring Logic: Adjust the scoring logic in the ViewController.swift file.
